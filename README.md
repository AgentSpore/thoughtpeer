# ThoughtPeer

> AI-powered journal with local LLM analysis and anonymous peer matching. Users write diary entries about their problems, local on-device AI analyzes patterns, and the server anonymously matches similar experiences and surfaces solutions from people who resolved similar issues. Privacy-first: only anonymized tags and categories are shared, never raw text.
